📋 Apps Script Setup Guide

🔗 Single Sheet for All Users
DoveCRM uses one shared Google Sheet for your entire team. All telecallers, managers and the super admin share the same Apps Script URL. Each user logs in with their PIN — data is synced centrally. You only need to set this up once, then share the same URL with all users.
Step 1: Go to script.google.com → New Project
Step 2: Click the 📋 Copy Code button below → Paste into the editor
Step 3: Click Deploy → New Deployment → Web App
Step 4: Set Execute as: Me · Who has access: Anyone
Step 5: Copy the Web App URL → Paste above → Save & Test
Step 6: Share the same URL with all team members (paste in their Settings)
